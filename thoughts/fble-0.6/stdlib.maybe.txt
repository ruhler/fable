What's the equivalent to ForEach for maybe?
  Maybe a -> b -> ((a, b) -> b) -> b

It's a little weird, because you know what 'b' is from the start if the
function is called. You could pass that to the function directly. So you could
have:

  Maybe a -> b -> (a -> b) -> b

Which is the 'maybe' function from Haskell, just in a different order. I like
the function last so we can use bind syntax if we want. Maybe better to put
default first, so:

  b -> Maybe a -> (a -> b) -> b

Which says: maybe we will apply a function, or maybe not. What it looks like:
  
  Int@ z = Maybe(3, mx, Incr);

Why wouldn't you just write: Default(3, Map(mx, Incr))? That makes more sense
to me.

Let's see where I'm using these funny For and DoWhile things.

DoWhile is used in sudoku to iterate over all possible symbols. It's like a
ForEach symbol, but with ForEach split into IterateSymbol. Basically a general
purpose iterator over a space of possible values.

For is used to iterate over the set of live aliens. I think that's a misuse.
That we should be doing ForEach over the set of aliens, not turning it into a
function to return the maybe next alien given the previous one. Though this
does raise a point about how to break out of for loops early.

Use of Maybe@ for DoWhile and For here is like an iterator that can compute
it's next value based on its current value, as opposed to having it baked in.
But we could just as well bake it in, by passing the current value to the next
value. I rather do this with 'Iter@' type than with Maybe@ type, if we even
want to have iterators of this style.

I vote we ditch For and DoWhile. Leave them in core for now. Phase them out
more properly.

Maybe and ForEach, we said, are basically equivalent. You might want ForEach
as a drop in replacement for the list, maybe, container equivalent. I think
that's worth having, and it makes sense to call it ForEach.

  maybe: Maybe a -> b -> (a -> b) -> b
  foreach: Maybe a -> b -> ((a, b) -> b) -> b

  Bool@ b := IfNothing(mv, 2);
  b.?(true: 1, false: 0);

  Int@ r = {
    Bool@ b := For(mv, 2);

  Int@ r = /Std/Maybe%.For(mv, 2, (Bool@ b) {

      b.?(true: 1, false: 0);
