fble-deps - generates depsfile for an fble module

Usage: .sp ffbbllee--ddeeppss [_O_P_T_I_O_N...] --tt _T_A_R_G_E_T --mm _M_O_D_U_L_E___P_A_T_H



Outputs  a  depfile  suitable for use with make and ninja specifying the ..ffbbllee
files the given module depends on.


Generic Program Information:
  --hh, ----hheellpp
      display this help text and exit
  --vv, ----vveerrssiioonn
      display version information and exit


Module Input:
  --II _D_I_R
      add _D_I_R to the module search path
  --pp, ----ppaacckkaaggee _P_A_C_K_A_G_E
      add _P_A_C_K_A_G_E to the module search path
  --mm, ----mmoodduullee _M_O_D_U_L_E___P_A_T_H
      the module path of the input module

  Packages are searched for in  the  package  search  path  specified  by  the
  FFBBLLEE__PPAACCKKAAGGEE__PPAATTHH  environment variable followed by the system default pack-
  age search path. Modules are searched for in the module search path.


  For example, if FFBBLLEE__PPAACCKKAAGGEE__PPAATTHH is //ffbbllee//ppkkggss and you provide command line
  options  --pp  ccoorree --mm //CCoorree//UUnniitt%%, the --pp ccoorree option adds //ffbbllee//ppkkggss//ccoorree to
  the module search path, and the --mm //CCoorree//UUnniitt%% option will look for the mod-
  ule at //ffbbllee//ppkkggss//ccoorree//CCoorree//UUnniitt..ffbbllee.


  The system default package search path is /usr/local/share/fble.



Output Control:
  --tt, ----ttaarrggeett _T_A_R_G_E_T

      target name to use in generated depfile




Exit Status:
  0
      Success.
  1
      Failure.
  2
      Usage error.

  Exit  status is 0 in the case where the .fble file cannot be parsed success-
  fully, to support the use case of generating dependencies  for  a  malformed
  .fble file.



Examples:

  ffbbllee--ddeeppss --tt HHeellllooWWoorrlldd..ffbblldd..dd --pp ccoorree --mm //CCoorree//SSttddiioo//HHeellllooWWoorrlldd%%
