fble-perf-profile - produces fble profile report for linux perf profile

Usage: ffbbllee--ppeerrff--pprrooffiillee [_O_P_T_I_O_N...]

Produces an fble profile report for a linux perf profile passed on stdin.


To collect perf data:



perf record -F 997 -d -g <cmd>



To report perf data:



perf report -q -g folded,count,0 | fble-perf-profile



The  output  ccoouunntt  field is mostly meaningless. It is the number of different
sample traces the function appeared in.
