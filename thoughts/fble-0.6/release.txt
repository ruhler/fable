Fble 0.6 Release
================

It's fairly late in the release development cycle now. How to finish out this
release?

The main thing for this release is standard library cleanup. I'm already
punting documentation for the standard library to later, because it's clearly
too much work. The standard library cleanup is a slog.

Other notable things I've changed along the way for the standard library
cleanup:
* Foreign Function Interface
* Composite character literal support (?)
* Change -> to := for monadic bind
* Support for '$' function application.

There's still a long way to go with the rest of the standard library cleanup.
