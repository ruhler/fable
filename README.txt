Directory Structure
-------------------
fblc/ - Source code for an fblc interpreter implemented in C.
fbld/ - Source code for an fbld interpreter implemented in C.
out/ - Directory where build outputs are placed, created as needed.
prgms/ - Directory containing sample fblc programs.
langs/ - Specifications of the fable languages.

Build and Test
--------------
To build and test everything, run:

  $ make

The results are placed in the out/ directory.
